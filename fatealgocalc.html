程序概述

本程序用于辅助命运算法终局结算，可以节省人力并减少计算错误率。

在代码框内输入若干行，每行代表一个变量的定义（可以插入空行）。程序会自动计算所有变量的值。

每行格式：变量名 = N ⊕ N ⊕ ……，其中 N 为数字或变量名，⊕ 为算符。

每两个符号之间必须添加空格，否则程序将无法正确识别算式。

变量名

变量名只能由一个或多个英文字母构成，不区分大小写（程序会自动把变量名转化成大写）。

引用变量名时，只能引用上方的变量，不能引用下方变量、本行变量或未定义的变量。

一般最后一行的变量名是 FINAL。

数字

数字只能输入整数（可以是负数）。若在数字后添加^2或²，则数字会被平方。

算符

算符包括 %、√（sqrt）、+、−（-）、×（*）、÷（/），每个算符代表一种二元运算。

括号内的形式不是标准形式，但程序也能识别，并更容易在标准键盘上打出。

若在算符左右添加括号，则表示这个算符的优先级最高。

算符优先级

算符的默认优先级是：“%” = “√” = “×” = “÷” > “+” = “−”。相同级别的运算顺序为从左到右。

若打开“同级运算”开关，则改为 “%” = “√” = “×” = “÷” = “+” = “−”。

计算结果

若输入格式无误，程序会计算每个变量的数值、（向 0 取整后的）位数、（向 0 取整后的）数根。

名为 FINAL 的变量结果会用粗体显示。
